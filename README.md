# AQIROX

Intelligent SaaS for Indian businesses. Marketing site for AQIROX, built with React, Vite, Tailwind CSS v4, and Motion.

## Local development

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:3000`.

## Production build

```bash
npm run build
```

Output is written to `dist/`. Preview it with `npm run preview`.

## Deploying to Vercel

This project is preconfigured for Vercel — `vercel.json` rewrites all routes to `index.html` so the custom SPA router handles client-side navigation.

1. Push this repo to GitHub.
2. Import the repo in Vercel — framework preset auto-detects **Vite**.
3. Build command: `npm run build`. Output directory: `dist`.
4. Deploy.

## Pages

- `/` — Home
- `/about` — About AQIROX
- `/careers` — Open roles
- `/contact` — Contact form
